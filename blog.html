Streamlit:

Streamlit is a tool meant to help create website in seconds, it takes very few lines of code to create amazing websites. There also HTML and CSS integrations that make cutstomatization extremely easy and today I'm gonna over how to use it.

Getting Started:

To initialize streamlit, the first thing you need to do is make sure python is installed.

  • Have a code editor open (I'm using vs code) that can run python and terminals
  • run the command "pip install streamlit"

      • This installs the streamlit library onto your computer so you can actually get starting using streamlit

  • Check out some of the public projects available to get a sense of what you are going to be able to do
  • When working with streamlit keeping this cheat sheet handy helps

Getting you first website up

To use streamlit you need to import it into your code editor

  • import streamlit as st - this allows us to call any action of the streamlit library by starting the command with st instead of typing out Streamlit

    • You will see almost all tutorial use "st" when referring to streamlit. You can name it whatever you want but the standard convention is st
